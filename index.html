🛠 Guide de configuration Firebase
À faire une seule fois — 5 minutes
1
Allez sur console.firebase.google.com et connectez-vous avec votre compte Google
2
Cliquez «Créer un projet» → nommez-le (ex : prodtrack) → désactivez Google Analytics → Créer
3
Dans le menu gauche → Build → Realtime Database → Créer une base de données → choisissez votre région → Démarrer en mode test
4
Retournez à l'accueil du projet → cliquez l'icône </> (Web) → enregistrez l'app → copiez l'objet firebaseConfig
5
Ouvrez ce fichier HTML dans un éditeur de texte (ex: Notepad) → remplacez les valeurs VOTRE_... dans la section firebaseConfig par vos vraies valeurs → sauvegardez
✓
Partagez le fichier HTML avec vos employés (via email, WhatsApp, clé USB…). Chacun l'ouvre dans son navigateur et remplit son rapport → vous voyez tout instantanément !
💡 Astuce : Pour un lien URL accessible partout, glissez ce fichier sur app.netlify.com/drop — gratuit et instantané.
